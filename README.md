# 🎴 卡牌對戰紀錄追蹤器

一個用於記錄卡牌遊戲對戰歷史的網頁應用程式，支援資料輸入、搜尋篩選與統計視覺化功能。

## 📥 下載與安裝

### 方法一：從 GitHub 下載
1. 點擊頁面右上方的綠色 **「Code」** 按鈕
2. 選擇 **「Download ZIP」**
3. 解壓縮下載的 ZIP 檔案到任意資料夾
4. 雙擊開啟 `index.html` 即可使用！

### 方法二：使用 Git Clone
```bash
git clone https://github.com/你的帳號/card-battle-tracker.git
cd card-battle-tracker
```

> 💡 **提示**：此程式完全在本機運行，不需要安裝任何軟體或網路連線！

## ✨ 功能特色

### 📝 紀錄輸入
- **類似 Excel 的表格** - 直覺的資料輸入介面
- **日期自動格式化** - 支援多種日期格式輸入，自動轉換為 `YYYY/MM/DD`
- **牌組自動完成** - 輸入過的牌組名稱會自動記憶
- **自訂欄位** - 可動態新增 T/F 類型的自訂欄位
- **編輯與刪除** - 隨時修改或移除紀錄

### 🔍 搜尋統計
- **多條件篩選** - 日期範圍、牌組、先後手、勝負等
- **統計指標**
  - 總場次
  - 先手率
  - 勝率
  - 先手勝率 / 後手勝率
- **視覺化圖表**
  - 分數變化曲線圖
  - 對手牌組分布圓餅圖

### 💾 資料管理
- **本地儲存** - 資料保存在瀏覽器 LocalStorage，無需網路連線
- **匯出功能** - 支援 JSON 和 CSV 格式匯出
- **匯入功能** - 可匯入 JSON 或 CSV 檔案

## 🚀 使用方式

### 方法一：直接開啟
雙擊 `index.html` 檔案即可在瀏覽器中開啟使用。

### 方法二：本地伺服器
如果需要完整功能，可以使用本地伺服器：

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx serve .
```

然後在瀏覽器開啟 `http://localhost:8080`

## 📋 欄位說明

| 欄位 | 類型 | 說明 |
|------|------|------|
| 對戰日期 | 日期 | 自動格式化為 `YYYY/MM/DD` |
| 己方牌組 | 文字 | 您使用的牌組名稱 |
| 對手牌組 | 文字 | 對手使用的牌組名稱 |
| 先後手 | 選項 | 先手 / 後手 |
| 勝負 | 選項 | 勝利 / 敗北 |
| 當前分數 | 數字 | 對戰後的分數 |
| 渣操等級 | 選項 | 嚴重 / 中等 / 輕度 / 無 |
| 渣操備註 | 文字 | 渣操的詳細描述 |
| 備註 | 文字 | 其他備註內容 |
| 自訂欄位 | T/F | 可自行新增的布林欄位 |

## 📅 支援的日期格式

程式會自動識別並轉換以下日期格式：

- `2026/01/07` ✓
- `2026-01-07` ✓
- `2026.01.07` ✓
- `20260107` ✓
- `01/07/2026` ✓
- `1/7/2026` ✓

## 💡 使用技巧

1. **快速輸入牌組名稱** - 輸入時會顯示之前使用過的牌組，可直接選取
2. **渣操備註** - 滑鼠懸停在表格中的渣操欄位可查看詳細備註
3. **備份資料** - 定期使用「匯出 JSON」功能備份您的紀錄
4. **跨裝置同步** - 可透過匯出/匯入功能在不同電腦間同步資料
